# xcode-product
 Launch xcode products (build results) from within emacs.

 This script depends on ivy https://github.com/abo-abo/swiper

 Usage:
 ```
 (require 'xcode-product)
 (global-set-key (kbd "f6") 'xcode-product-launch)

 ```
